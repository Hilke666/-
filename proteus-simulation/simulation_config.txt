## 仿真提示
[Tips]
1. 运行仿真前确保所有元件已正确连接
2. 使用"Probe"工具查看各节点的电压值
3. 使用"Ammeter"工具查看电路电流
4. LED正极接正电压，负极接地
5. 220Ω电阻确保LED工作在安全的电流范围内
6. 可以双击元件修改其参数值
7. 使用快捷键 Ctrl+R 快速启动/停止仿真

## 配置步骤
# 在Proteus中配置这些参数的方法：
# 1. 点击"Simulation" → "Edit Simulation Model"
# 2. 在"Transient"标签页中配置时间参数
# 3. 在各个元件属性中设置具体数值
# 4. 点击"Run"开始仿真
