## 🧮 电路计算

### 电流计算
```
I = (VCC - Vf) / R1
I = (5V - 2.0V) / 220Ω
I = 3V / 220Ω
I = 13.64mA ✓
```

### 功率计算
```
P_R1 = I² × R1 = (0.01364)² × 220 = 40.9mW
P_LED = Vf × I = 2.0V × 0.01364A = 27.3mW
P_Total = VCC × I = 5V × 0.01364A = 68.2mW
```

### 安全性检查
✅ LED电流 (13.64mA) < 最大值 (30mA)  
✅ 电阻功率 (40.9mW) < 1/4W (250mW)  
✅ 所有参数在安全范围内
