# Proteus 8.0 简单LED电路仿真

## 📋 项目说明
这是一个简单的LED闪烁电路仿真项目，可以直接在Proteus 8.0中打开和运行。

## 🔌 电路组成
- **电源**：5V VCC
- **电阻**：220Ω（限流电阻，保护LED）
- **LED**：红色发光二极管
- **地**：GND（电路的参考点）

## 📐 电路原理
```
VCC (5V) ──── R1(220Ω) ──── D1(LED) ──── GND
```

电流流向：
1. 从5V电源出发
2. 通过220Ω电阻降压限流
3. 通过LED使其发光
4. 流回地（GND）

## 🚀 如何使用

### 方法一：打开现成的项目文件
1. 打开Proteus 8.0
2. 点击 `File` → `Open Project`
3. 选择 `LED_Circuit.pdsprj` 文件
4. 点击工具栏的绿色"播放"按钮启动仿真

### 方法二：导入原理图文件
1. 打开Proteus 8.0，新建项目
2. 点击 `File` → `Open` 
3. 选择 `LED_Circuit.dsn` 文件
4. 配置仿真参数（见下方）

## ⚙️ 仿真参数建议

- **仿真时间**：1秒
- **时间步长**：0.1ms
- **电源电压**：5V（±5%容差）
- **LED规格**：红色，Vf≈2V，If≈20mA

## 🧮 理论计算

### 限流电阻计算
```
R = (VCC - Vf) / If
R = (5V - 2V) / 20mA
R = 3V / 0.02A
R = 150Ω (选择标准值220Ω)
```

### 实际LED电流
```
I = (5V - 2V) / 220Ω ≈ 13.6mA
```

## ✅ 预期结果
仿真运行时，LED应该显示红色发光（在Proteus中会亮起）。

## ❓ 常见问题

**Q1：LED不亮怎么办？**
- 检查LED极性是否正确（阳极接正，阴极接负）
- 检查电源是否连接正确
- 确认电阻值不要过大（建议220Ω以下）

**Q2：能否添加闪烁效果？**
- 可以！需要在电路中添加555定时器或单片机生成PWM信号
- 参考进阶电路文件：`LED_Blink_PWM.dsn`

**Q3：如何修改电源电压？**
- 双击VCC元件，在弹出的对话框中修改电压值
