# Proteus 8.0 LED电路仿真 - 专业版

## 📦 工程文件

本项目包含可直接在 **Proteus 8.0 专业版** 中打开的工程文件。

### ✅ 主要文件

| 文件 | 类型 | 用途 |
|------|------|------|
| **LED_Circuit.dpj** | Proteus项目 | 直接打开使用 |
| **LED_Circuit_Pro.xml** | XML配置 | 备用项目 |

### 📄 配置文件
- LED_Circuit.sch - 原理图配置
- LED_Circuit.pro - 工程配置清单

### 📖 文档
- README_PROFESSIONAL.md - 详细说明
- QUICK_START_PRO.md - 快速开始

---

## 🚀 立即使用

```
1. File → Open Project
2. 选择 LED_Circuit.dpj
3. 点击绿色 ▶ 运行仿真
```

---

## 📊 电路信息

- **电源**：5V DC
- **限流电阻**：220Ω
- **LED**：红色发光二极管
- **工作电流**：13.6mA
- **总功耗**：68.2mW
