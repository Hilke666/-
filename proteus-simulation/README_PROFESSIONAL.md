# Proteus 8.0 专业版 LED电路仿真项目

## 📦 工程文件说明

本文件夹包含可以直接在Proteus 8.0专业版中打开的工程文件。

### ✅ 可直接打开的文件

| 文件名 | 格式 | 用途 | 打开方式 |
|--------|------|------|----------|
| **LED_Circuit.dpj** | Proteus项目 | ⭐ 推荐 | File → Open Project |
| **LED_Circuit_Pro.xml** | XML格式 | 备用 | File → Open |
| **LED_Circuit.pro** | 配置文件 | 参考 | 文本编辑器 |
| **LED_Circuit.sch** | 配置文件 | 参考 | 文本编辑器 |

---

## 🚀 快速开始

### 方法1：打开项目文件（推荐）
```
1. 启动 Proteus 8.0 专业版
2. File → Open Project
3. 选择 LED_Circuit.dpj
4. 点击打开
```

### 方法2：导入XML文件（备用）
```
1. 启动 Proteus 8.0 专业版
2. File → Open
3. 选择 LED_Circuit_Pro.xml
4. 点击打开
```

---

## ▶️ 运行仿真

打开工程后：

```
1. 点击工具栏的绿色 ▶ 按钮
   或 按 Ctrl+R
   或 Simulation → Run

2. 观察LED的发光效果（应显示为红色）

3. 使用 Probe 工具测量各节点的电压和电流

4. 仿真自动运行1秒后停止
```

### 元件列表

```
┌─────────────────────────────────────────┐
│  U1: VCC (5V DC 电源)                   │

           ↓

└─────────────────────────────────────────┘
           ↓
┌─────────────────────���───────────────────┐

           ↓
```

### 仿真参数

- **分析类型**：Transient (瞬态分析)
- **运行时间**：0 → 1秒
- **时间步长**：0.1ms
- **最大步长**：1ms
- **收敛容限**：1e-6
- **自动时间步**：启用

## 📝 使用建议

### 第一次使用
1. 直接打开 LED_Circuit.dpj
2. 点击绿色播放按钮运行
3. 观察LED亮起

### 修改参数测试
1. 双击 R1 修改电阻值
2. 双击 U1 修改电源电压
3. 双击 D1 修改LED类型
4. 重新运行仿真

### 查看波形
1. 仿真运行中，点击 Tools → Oscilloscope
2. 点击节点添加测量点
3. 观察电压/电流波形

## 🔧 故障排查

### 问题：无法打开文件
**解决**：
- 确保是Proteus 8.0或更高版本
- 尝试用 LED_Circuit_Pro.xml 替代
- 检查文件是否完整

### 问题：LED不亮
**解决**：
- 检查LED极性（长脚为正）
- 检查电阻值不要过大
- 用Voltage Probe测量LED两端电压

### 问题：仿真错误
**解决**：
- Simulation → Reset All
- 关闭并重新打开项目
- 参考 TROUBLESHOOTING.md
