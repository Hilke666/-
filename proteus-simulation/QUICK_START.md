# 快速开始 - Proteus 8.0 专业版

## ⚡ 10秒开始仿真

### 步骤1：打开
```
File → Open Project → LED_Circuit.dpj → Open
```

### 步骤2：运行
```
点击 ▶ 或按 Ctrl+R
```

### 步骤3：完成
```
LED显示红色 = 成功 ✅
```

---

## 📊 预期结果

| 项目 | 值 |
|------|-----|
| VCC | 5.0V |
| R1压降 | 3.64V |
| LED压降 | 2.0V |
| 电流 | 13.6mA |
| LED状态 | 亮(红色) |
