# 🛠️ Proteus 8.0 LED电路 - 手动创建指南\n\n## ⚠️ 注意\nXML格式的.dsn文件可能无法被Proteus直接打开。\n**本指南提供了两种解决方案：**\n\n---\n\n## 方案A：在Proteus中手动创建（推荐）\n\n这是最快、最可靠的方法。只需5分钟！\n\n### 步骤1：新建项目\n```\n1. 打开 Proteus 8.0\n2. File → New Project\n3. 输入项目名称：\"LED_Circuit\"\n4. 选择 \"Create a schematic from template\"\n5. 点击 Finish\n```\n\n### 步骤2：添加元件\n\n在原理图编辑界面，点击左侧工具栏的\"P\"按钮（Pick Device）：\n\n#### 添加VCC电源\n```\n1. 点击 P (Pick Device)\n2. 搜索框输入：VSRC\n3. 选择 VSRC (Voltage Source)\n4. 点击到画布上放置\n5. 双击，设置参数：\n   - Value: 5\n   - Unit: V\n```\n\n#### 添加电阻R1\n```\n1. 点击 P (Pick Device)\n2. 搜索框输入：RES\n3. 选择 RES (Resistor)\n4. 点击到画布上放置\n5. 双击，设置参数：\n   - Value: 220\n   - Unit: Ω\n```\n\n#### 添加LED\n```\n1. 点击 P (Pick Device)\n2. 搜索框输入：LED\n3. 选择 LED (发光二极管，颜色RED)\n4. 点击到画布上放置\n5. 双击，设置参数：\n   - Value: RED\n   - Vf: 2.0V\n   - If: 20mA\n```\n\n#### 添加地GND\n```\n1. 点击 P (Pick Device)\n2. 搜索框输入：GND\n3. 选择 GND (Ground)\n4. 点击到画布上放置（两个地点）\n```\n\n### 步骤3：连接导线\n\n使用工具栏中的\"Wire\"工具（或按W快捷键）连接：\n\n```\n连接顺序：\n1. VCC正极 → R1一端\n2. R1另一端 → LED正极(长脚)\n3. LED负极(短脚) → GND\n4. VCC负极 → GND\n```\n\n**连接技巧：**\n- 每次点击开始点，然后点击结束点\n- 按 ESC 退出连线模式\n- 导线交叉时自动连接（显示小黑点）\n\n### 步骤4：配置仿真参数\n\n```\n1. 点击菜单 Simulation → Edit Simulation Model\n2. 点击 \"Transient\" 标签\n3. 设置参数：\n   - Start Time: 0\n   - End Time: 1s\n   - Step Size: 0.0001s (或 0.1ms)\n   - 其他默认即可\n4. 点击 OK\n```\n\n### 步骤5：运行仿真\n\n```\n1. 点击菜单 Simulation → Run\n   或 按 Ctrl+R\n   或 点击工具栏的绿色 ▶ 按钮\n2. 观察LED是否发光（应该显示为红色）\n3. 使用 Probe 工具查看电压和电流值\n```\n\n### 步骤6：保存项目\n\n```\n1. File → Save Project\n2. 输入文件名\n3. 选择保存位置\n4. 点击 Save\n\n生成的文件：\n- 项目文件：*.pdsprj\n- 原理图文件：*.dsn\n- 备份文件：*.bak\n```\n\n---\n\n## 方案B：使用现成的参考文件\n\n本仓库提供了以下参考文件，您可以参照其中的参数和连接方式：\n\n### 📄 参考文件\n- `simulation_config.txt` - 仿真参数\n- `Circuit_Analysis.md` - 电路分析和计算\n- `README.md` - 项目说明\n\n### 使用步骤\n1. 打开这些文档\n2. 根据参数手动在Proteus中创建\n3. 完成后保存您自己的项目文件\n\n---\n\n## 完整电路图参考\n\n```\n原理图连接图：\n\n        ┌─────────────┐\n        │  VCC (5V)   │\n        │   (+)    (-) │\n        └─────┬───────┘\n              │\n              ▼\n         ┌────────────┐\n         │  R1 220Ω   │\n         │  ────────  │\n         └─────┬──────┘\n               │\n               ▼\n         ┌────────────┐\n         │  D1 (LED)  │ ◀── 红色LED\n         │   (+) (-)  │     长脚(+)\n         │    ▲   ▼   │     短脚(-)\n         └────┼───┼───┘\n              │   │\n              │   └────┐\n              │        │\n              ▼        ▼\n        ┌─────────────────┐\n        │ GND (地)        │\n        │ 参考点(0V)     │\n        └─────────────────┘\n\n电流路径：\nVCC(+) → R1 → LED(+) → LED(-) → GND(-) → VCC(-)\n```\n\n---\n\n## 元件参数清单\n\n### 📍 位置建议\n```\nVCC:   X=100, Y=50\nR1:    X=150, Y=50\nD1:    X=250, Y=50\nGND:   X=100, Y=150 和 X=250, Y=150\n```\n\n### 📊 参数表\n\n| 元件 | 类型 | 值 | 参数 |\n|------|------|-----|------|\n| VCC | VSRC | 5V | DC电源 |\n| R1 | RES | 220Ω | ±5%误差 |\n| D1 | LED | RED | Vf=2.0V, If=20mA |\n| GND | GND | - | 参考地 |\n\n---\n\n## 仿真参数速查表\n\n| 参数 | 值 | 说明 |\n|------|-----|------|\n| 分析类型 | Transient | 瞬态分析 |\n| 开始时间 | 0s | 从0开始 |\n| 结束时间 | 1s | 运行1秒 |\n| 时间步长 | 0.1ms | 每步0.1ms |\n| 最大步长 | 1ms | 不超过1ms |\n| 容限 | 1e-6 | 收敛精度 |\n\n---\n\n## 测试清单\n\n仿真运行后，检查以下项目：\n\n- [ ] LED显示为红色（亮起）\n- [ ] VCC节点电压约 5V\n- [ ] R1两端压降约 3.64V\n- [ ] LED两端压降约 2.0V\n- [ ] 电路电流约 13.6mA\n- [ ] 仿真时间显示从0到1s\n- [ ] 无错误警告信息\n\n---\n\n## 常见问题\n\n### Q1: 如何添加标签？\n```\n1. 点击工具栏中的 \"T\" (Text)\n2. 点击要标注的位置\n3. 输入文本（如\"5V\", \"R1\", \"LED\"）\n4. 点击OK\n```\n\n### Q2: 如何旋转元件？\n```\n1. 点击选中元件\n2. 按 R 键旋转\n3. 或右键 → Rotate\n```\n\n### Q3: 如何删除错误的元件或导线？\n```\n1. 点击选中要删除的对象\n2. 按 Delete 键\n3. 或右键 → Delete\n```\n\n### Q4: 仿真后如何保存波形？\n```\n1. 仿真运行中，点击 Tools → Instruments → Oscilloscope\n2. 探针测量的结果会显示在示波器中\n3. File → Save 保存波形数据\n```\n\n---\n\n## 预期结果\n\n### 仿真运行时应该看到：\n1. ✅ LED显示为亮红色\n2. ✅ 窗口下方显示 \"Simulation running...\"\n3. ✅ 时间计数器从 0 → 1s\n4. ✅ 无红色错误提示\n\n### 用Probe工具测量应该看到：\n1. ✅ VCC = 5.0V\n2. ✅ R1压降 = 3.64V\n3. ✅ LED压降 = 2.0V\n4. ✅ 电流 = 13.6mA\n\n---\n\n## 下一步\n\n完成基础LED电路后，您可以尝试：\n\n1. **添加开关**：在VCC和R1之间加入开关\n2. **多LED电路**：并联或串联多个LED\n3. **555定时器**：实现LED闪烁\n4. **PWM调光**：用单片机控制亮度\n5. **传感器应用**：光敏二极管、温度等\n\n---\n\n## 技术支持资源\n\n- 📖 Proteus官方文档：https://www.labcenter.com/\n- 🎓 Proteus教程：在Help菜单中搜索\n- 🔧 组件库：系统默认库包含所有常用元件\n- 💬 用户论坛：https://www.labcenter.com/forum/\n\n---\n\n**祝您仿真顺利！如有问题请参考 TROUBLESHOOTING.md** 🎉\n\n创建日期：2026-06-27  \n适用于：Proteus 8.0 及以上版本  \n难度等级：⭐ 初级"
