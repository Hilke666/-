# 快速指南 - Proteus 8.0专业版

## ⚡ 10秒打开仿真

### 步骤1：打开文件
```
File → Open Project → 选择 LED_Circuit.dpj → Open
```

### 步骤2：运行仿真
```
点击绿色 ▶ 按钮 或 按 Ctrl+R
```

### 步骤3：观察结果
```
LED显示为红色 = 成功! ✅
```

---

## 📊 预期值

```
VCC = 5.0V
R1压降 = 3.64V
LED压降 = 2.0V
电流 = 13.6mA
LED = 亮(红色)
```

## 🛠️ 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+R | 运行/停止仿真 |
| Ctrl+L | 添加标签 |
| R | 旋转元件 |
| Delete | 删除 |
| Ctrl+Z | 撤销 |
| Ctrl+S | 保存 |
